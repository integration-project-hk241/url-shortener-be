# Prerequisites

Make sure you installed [node](https://nodejs.org), [Docker](https://www.docker.com/) and [Docker compose](https://docs.docker.com/compose/) on your machine.

## Project setup

```bash
$ npm install
$ docker compose up
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
